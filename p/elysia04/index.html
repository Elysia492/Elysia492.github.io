6.vi-vim(文本编辑器)

1.打开文件（不存在就自动创建这个文件）

vim + 文件名

1
2
3
4
5

[root@elysia ~]# ls
anaconda-ks.cfg  elysia  webtatic-release.rpm
[root@elysia ~]# vim text.txt
[root@elysia ~]# ls
anaconda-ks.cfg  elysia  text.txt  webtatic-release.rpm

vim + 目录/文件

1
2
3
4
5
6

[root@elysia ~]# ls -l elysia/
total 0
[root@elysia ~]# vim elysia/text.txt
[root@elysia ~]# ls -l elysia/
total 4
-rw-r--r-- 1 root root 3 May 31 05:10 text.txt

若目录不存在，无法创建，会在保存退出的时候出错

1
2
3

[root@elysia ~]# ls
anaconda-ks.cfg  elysia  text.txt  webtatic-release.rpm
[root@elysia ~]# vim 111/text.txt

1
2
3

Press ENTER or type command to continue

2.按字母 i 进入编辑模式

3.按Esc推出编辑

4.输入

  • :wq 保存退出

  • :q! 强制退出不保存

  • :wq! 强制退出并保存
