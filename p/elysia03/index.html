Linux目录结构初级与Linux极其核心命令

2026-03-17

1. Linux核心目录

根下的核心目录 说明
/etc/ 系统服务配置文件的目录，存放系统，服务，命令的配置。
/home/ 普通用户家目录
/root/ root用户家目录
/dev/ device,设备文件目录，硬盘，光盘
/tmp/ temp临时目录，临时存放内容
/proc/ process 系统服务，进程信息，类似于汽车的仪表盘

2.路径表示形式

绝对路径：从根目录（ / ）出发

相对路径：从当前目录出发

3.Linux命令

1.cd ,pwd

  • cd：回到家目录

  • cd ~：回到家目录

  • cd ..：回到上级目录 (cd ../../回到上上级目录)

  • pwd：查看当前位置

2.mkdir 创建目录

mkdir elysia

mkdir -p elysia/elysia1/elysia2 多层创建目录

3.touch 创建文件

touch elysia

创建elysia文件发现问题，ai说文件和目录同名，只会更新目录访问和修改时间

创建elysia1，创建成功

创建两个文件需要用绝对路径

如果第一个用绝对路径 ，第二个没用， 第二个文件按相对路径创建文件

4.ls查看目录

ls elysia

ls -l elysia 可以简写为ll elysia

ls -l -t elysia 按照时间查看目录内容，选项可合并ls -lrt elysia

ls -l -t -r root 按照时间查看目录内容并逆序排序，选项可合并ls -lrt elysia

ls -lh elysia（ll -h elysia） 让文件大小变为人类可读形式

4096变为4.0K

  • 1KB=1024bytes
  • 1MB=1024KB
